Método de uso: 
1- La aplicación web inicia en la página principal. En ella pude seleccionar filtrar los productos (en este caso mangas japoneses)
  por su editorial. Además, dispone de una barra de búsqueda de manga por nombre. El nombre tiene que ser escrito igual que como está en las tarjetas.
  Por último, se dispone de un widwet que redirige al usuario al carrito de compras

2- Cada manga tiene su propia tarjeta con su portada correspondiente, el nombre y el precio. Al hacer click en una la página se dirige a 
  la página con los detalles del producto. En esta página se disponen de 2 botones, 1 para volver a la página anterior, sea cual sea, y él 
  otro para agregar el producto al carrito junto con un contador. Una vez presionado, el botón agregará el producto con la cantidad indicada 
  en el contador y deshabilitará el botón para que el usuario no pueda agregar más de ese producto. En caso de no tener stock del producto 
  aparecerá un botón de "SIN STOCK" para que el usuario no pueda interactuar con el botón ni con el contador para agregar el producto. Si 
  quiere cambiar la cantidad o quitarlo del carrito podrá hacerlo en la página del carrito.

3- Una vez en el carrito se podrá editar los productos a voluntad. El usuario dispone de 2 botones, uno para volver a la página anterior y 
  agregar más productos y el otro para hacer un checkout de la compra. 

4- El check out solo es un formulario con los datos básicos para la compra. Al tocar el botón finalizar compra, la compra se cargará en la BASE
  de datos y se restarán los productos comprados del stock total. Aparecerá un cartel agradeciendo al usuario junto con la información que acaba 
  de cargar. Por último, el usuario podrá interactuar con un botón para volver a la página principal y seguir comprando.
